Ruboty::MessageSuffix
==========

## Overview
This Extension is for [Ruboty](https://github.com/r7kamura/ruboty).

Grant any word to suffix of message.



## Usage

```bash
export RUBOTY_MESSAGE_SUFFIX='どすぅ'
bundle exec ruboty
> ruboty ping
pong どすぅ

```

$ bundle

$ gem install ruboty-message_suffix
